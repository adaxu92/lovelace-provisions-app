# Lovelace Provisions
    - A daily forecasting web app based on user's horoscope sign.
    https://lovelace-provisions.herokuapp.com/

## Using:
 - Ruby on Rails
 - Twilio API
 - The Astrology API
 - PostgreSQL
 - HTTParty
 - Action Mailer

## Features:
 - User comments
 - Thumbs up and thumbs down
 - Today's & Yesterday's horoscope forecast
 - Text and email a friend
